EatCrow2

EatCrow2 is a full-stack SCADA platform that integrates a C++/Python serial backend with an interactive web dashboard for real-time autoclave control and telemetry visualization. It features a custom domain-specific scripting engine for automated curing profiles, hands-free voice controls, and heuristic diagnostic algorithms for hardware fault isolation.

C++ Arduino Python SCADA Control Theory

StarForge

StarForge is a custom domain-specific language (DSL) and macro interpreter designed to bridge C++ backend processing with JavaScript UI controls for real-time autoclave automation. It simplifies sterilizer operations by transpiling human-readable macro scripts into dynamic control sequences, variable logic, and live telemetry adjustments.
